In order to start, it is important to know how to deal with the Terminal. This also will give you a rough idea of what coding means.

Instructions will change depending on what OS you have.

If you have a Windows

  • cd and cd..
  • list and dir
  • opening files and history

If you have a Mac

  • all of the above

Things you absolutely need to know

  • Check the folder you are currently in with the terminal (pwd)
  • Move into another folder (cd + next folder)
  • Move back to the previous folder (cd ..)
  • Check what there is in a folder (ls or dir)
