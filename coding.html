What is the best way to learn to code?

There is no perfect answer to this, but in experience, the best way is to borrow other people's code and edit it until you understand it and it works for you (ask for demos). Often you do not have to code everything from scratch, but only change a bit of previous versions ;-) In addition, running tutorials (doing the exercises in real-time) can also be a very good way to familiarize yourself with the coding language.
