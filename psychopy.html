The lab uses PsychoPy to run online experiments.

PsychoPy is an open source software, written in the Python programming language, for the generation of experiments for neuroscience and experimental psychology. At this level you do not have to worry about coding, it is a G.U.I. interface (called Builder)!

The founder of PsychoPy (prof. Jon Peirce) has recently given a seminar about PsychoPy for local and online experiments: Team Talk link (April 2020)
